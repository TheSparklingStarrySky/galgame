# 《零点协议》音频规划

## 一、声音方向

音频不应把游戏做成持续轰鸣的恐怖片。常态以室内空气声、荧光灯、电气噪声和极简音乐维持压迫；规则公开、项圈警告、死亡与封锁才显著提高动态。主办方、PDA 与项圈共用一个冷硬的三音电子动机，让玩家在听见完整提示前就能预感系统正在介入。

现阶段已完成 **19 首功能型 BGM、62 个可复用音效/环境声**。P0 使用经来源留档的 Pixabay 素材，P1 的协作、夜间、三条恋爱线、互相残杀、审计主题及第五、第七章专属动作声使用项目原创程序合成资源。

详细文件名、优先级、循环属性与用途记录在 `assets/audio/audio_manifest.json`。

## 三、音效分层

### UI 与 PDA

`ui_confirm`、`ui_cancel`、`pda_open`、`pda_close`、`save_complete`、`load_complete`、`choice_reveal`、`route_jump`、`clue_acquired`、`combine_success`、`combine_fail`。

普通正文翻页不播放按钮音，避免数小时阅读中产生疲劳。只有明确操作、状态变化和奖励反馈发声。

### 主办方与项圈

`terminal_boot`、`pda_notification`、`administrator_channel_on/off`、`collar_warning`、`collar_lock`、`collar_detonation`、`facility_alarm`、`access_granted/denied`、`surveillance_servo`。

`collar_detonation` 不应是夸张炸弹声：先是贴近耳侧的电容爆裂和金属冲击，再接身体倒地；爆响之后立刻压低音乐，保留耳鸣或短暂静默。

### 场景环境

`amb_fluorescent_hum`、`amb_ventilation`、`amb_corridor_roomtone`、`amb_assembly_pa`、`amb_infirmary_equipment`、`amb_storage_refrigeration`。环境音在场景切换时 500-900ms 交叉淡化，同一空间连续对白不要重新起播。

### 剧情动作与解谜

包括金属门、脚步、扳手撞击、卷帘门、绳索受力、千斤顶滑脱、医疗监护、定向音频，以及拾取、钥匙开箱、紫外灯、卷尺、键盘、天平、石板滑动和机关通电。一次性关键物件尽量使用独立音色；普通物品拾取可以复用。

## 四、Pixabay 可用性

Pixabay 当前许可允许免费使用、无需强制署名、允许修改，并可用于商业或非商业的新作品。游戏内嵌 BGM/音效属于组合创作；禁止的是把原音频作为独立素材重新出售或分发。每个素材仍需逐条检查作者说明、第三方权利和 Content ID 状态。

推荐工作流：

1. 在素材详情页试听并确认存在 `Free for use under the Pixabay Content License`。
2. 优先排除标有 `Content ID Registered`、作者另行要求署名或说明中出现额外商用限制的曲目。
3. 下载时保存详情页 PDF/截图、作者、链接、原始文件名和日期到内部授权记录。
4. 保留原始 MP3 作为 source master，编辑后导出 48kHz WAV 母版和游戏用 OGG。
5. 检查循环接缝、响度和对白遮蔽；不要直接把下载文件改名后投入构建。

## 五、AI 生成平台

### 首选：ElevenLabs

同时提供 Music 与 Sound Effects。音效可指定 0.1-30 秒、循环和提示词影响程度，可下载 48kHz WAV；音乐可按段编辑，最长五分钟，并明确覆盖游戏用途。正式商用必须在允许商用的付费方案内生成，免费方案输出不要进入发布版。

适合：三条恋爱主题、主办方三音动机、项圈专属音效、无缝室内环境声。

### 次选：Stable Audio

能生成音乐和音效，训练数据说明为合作方 AudioSparx 提供。Creator License 允许个人创作者用于商业项目和游戏；若项目主体或用户规模超过其许可阈值，应升级或重新确认 Enterprise 条款。

适合：长氛围、工业噪声、循环脉冲、危机音乐草案。

### 可选：Adobe Firefly Audio

可生成最长约 30 秒的音效和环境声，也提供 Generate Soundtrack。适合通过时间轴叠加多层 Foley；当前部分能力仍标注 beta，下载前应保存当时产品页对商用的具体说明。

适合：门、卷帘、工具、医疗设备、组合机关等需要精确时序的短音效。

### 暂不作为主方案：Suno

付费订阅期间生成的音乐可取得商用权，但免费层只允许非商业用途；它更擅长完整歌曲，精确循环、短交互音效和统一游戏动机不如前三者。若使用，只做无歌词草案，不在提示词中模仿真实艺术家或现有作品。

## 七、制作顺序

1. 先做 `title_protocol` 的三音动机，并从它派生 PDA、主办方和项圈提示音。
2. 完成第一章 P0：苏醒、集合、处决、余波，加基础环境与门锁音效。
3. 完成调查/推理通用音乐和背包、组合、机关反馈。
4. 补体育馆与医疗危机音乐，再制作三条恋爱主题。
5. 全部剧情完成后才定稿审计真相与片尾曲，避免主题提前泄底。
